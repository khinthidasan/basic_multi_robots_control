A basic framework to control multiple robots using their provided APIs:

Dobot Arm
SEER AMR
Jibot AMR
PLC Machines

For advanced usage, you can integrate:
asyncio, threading, multiprocessing,
IoT concepts,
Computer vision modules
Route management etc.,
